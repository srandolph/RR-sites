Drupal Real Estate (DRE) is a solution for the creation of a real estate and property
classifieds sites.

Note: the module is under active development and for now is available only basic
functionality. Soon will be available much more.

INSTALLATION

Short description:

The module provides easy installation:
1. Download all required modules and libraries.
2. Enable the Kickstart module.
3. Go to a Real Estate menu.

Full description:

1. Download and enable required modules:
  Entity API - http://drupal.org/project/entity
  Views - http://drupal.org/project/views
  Address Field - http://drupal.org/project/addressfield
  CTools - http://drupal.org/project/ctools

  If you plan to use the Flex Slider module then also download:
  Flex Slider - http://drupal.org/project/flexslider
  Libraries API 2.x - http://drupal.org/project/libraries
  FlexSlider library from https://github.com/woothemes/FlexSlider and put a
  content of archive to the sites/all/libraries directory.
  For more details, please, follow the installation instruction in a section
  Installation of the page http://drupal.org/project/flexslider

  Then go to the Real Estate section and enable a DRE Kickstart module
  if you want use the whole DRE package.

  Instead using a Flex Slider module, can be used any other slider module.

  Instead using the DRE SimpleGMap module can be used more complex and
  powerful solution. Addressfield address can be automatically Geocoded into
  Geofield points for display on Openlayers Maps.

2. Go to the Real Estate menu (admin/real_estate) in the admin area.
  For adding property types go to admin/real_estate/property_type
  For add property go to admin/real_estate/property/add
  After adding property it will be accessible from a page /real-estate-property-list
